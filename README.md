# Library Management System Frontend

Frontend part of the **Library Management System** course project developed for managing a digital library collection.  
This application provides a user-friendly interface for authentication, catalog browsing, and administrative book management.

## Project Overview

The frontend is built as a modern single-page application using **Angular** and communicates with the backend REST API developed with **Java Spring Boot**.

The system supports two user roles:

- **USER** — can browse the catalog, search books, and view available information
- **ADMIN** — can manage the library collection, including adding, editing, and writing off books

## Features

- User registration and login
- JWT-based authentication
- Role-based access control
- Home page
- Book catalog page
- Admin dashboard
- Integration with backend REST API
- Responsive frontend architecture
- Route guards for protected pages
- HTTP interceptor for automatic JWT token injection

## Tech Stack

- **Angular**
- **TypeScript**
- **SCSS**
- **Angular Router**
- **Reactive Forms**
- **Angular HttpClient**
